For the following libraries, Servo-1.2.1 and roboclaw_arduino_library-master, download the .zip, keep it zipped, and follow these steps:

Open the Arduino IDE -> Sketch -> Include Library -> Add .zip Library -> Choose the .zip you wish to add

All done you have em.

For the following libraries, CameraControl, Movement, and PinDefinitions, download the folders and follow these steps:

Open your file explorer to where you downloaded the libraries
Right click on each of the libraries individually
Send to -> .zip
Once the file is in a .zip format repeat for the remaing libraries
After all are finished go into the 
Arduino IDE -> Sketch -> Include Library -> Add .zip Library -> Choose the .zip you wish to add
Make sure you add all of the libraries.
